# XSDProject
XML Schema for a type of XML document

## General Info

This project has implement an XML Schema(XSD) for a type of XML document that contains information about flights at the LONDON International Airport. 

## Technologies
* XML
* XSD
* HTML
* JavaScript
* CSS
